Radiant includes two AI workflows for docs. The writing agent helps your team
create and revise documentation. Ask AI helps readers ask questions about your
published docs.

| AI feature | Who uses it | What it does |
| --- | --- | --- |
| Writing agent | Your docs team | Drafts, edits, restructures, and cleans up docs |
| Ask AI | Your docs readers | Answers questions from published docs and approved context |

## Ask the writing agent for edits

### Give it a goal

Use the writing agent when you want to improve the docs themselves. Give it the
goal, the audience, and any product details it should preserve.

### Try specific requests

Good requests are specific:

- "Rewrite this page for a first-time user."
- "Add a short troubleshooting section for failed imports."
- "Create a quickstart from these notes."
- "Make this page easier to scan without removing important details."

## Prepare docs for Ask AI

### Write answerable pages

Ask AI works best when your docs are clear, structured, and direct. Write pages
that answer the questions readers are likely to ask.

### Include useful source content

Good Ask AI source content includes:

- Clear product terms and definitions.
- Step-by-step workflows.
- Expected outcomes after important actions.
- Status labels and what they mean.
- Concise FAQs for common decisions.

<Callout type="info" title="Write for people first">
  Good docs also make better AI answers. If a reader can find the answer on the
  page, Ask AI has a better chance of finding it too.
</Callout>
