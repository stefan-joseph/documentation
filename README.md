# Your Docs

This repository contains the source files for your Radiant Docs site.

## What is in this repo

- `docs.json` configures the site title, homepage, navigation, and assistant
  starter questions.
- `.mdx` files are documentation pages.
- Images, OpenAPI files, and other media can be added when your docs need them.

## How to edit

You can edit these docs in three ways:

- Use the Radiant dashboard for visual editing and publishing.
- Ask the writing agent to draft or revise pages.
- Edit the MDX files and `docs.json` directly.

## How to publish

When your changes are ready, publish them from Radiant. Radiant builds the docs
site and deploys the latest version for your readers.
